# Nibin Joseph - Portfolio

A personal portfolio website showcasing my professional experience, technical projects, and academic background.

## Tech Stack

- **Frontend**: [React 19](https://react.dev/), [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
- **Deployment**: [GitHub Pages](https://pages.github.com/)

## Project Structure

- `src/features`: Contains all major portfolio sections (About, Experience, Projects, etc.)
- `src/shared/components`: Reusable UI components like the Navbar, Footer, and ScrollToTop button.
- `src/core`: Core configurations and asset constants.

## Getting Started

To run the project locally:

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
